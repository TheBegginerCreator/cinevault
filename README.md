# CineVault 🎬

## О проекте

CineVault — это современный одностраничный сайт о кино, созданный в формате лендинга. Проект был разработан как итоговая работа по HTML, CSS и JavaScript. Основная задача — создать красивый, адаптивный и интерактивный сайт с каталогом фильмов, отзывами пользователей и формой обратной связи.

Сайт поддерживает светлую и тёмную тему, содержит динамическую загрузку карточек фильмов и адаптирован под мобильные устройства.

---

## Цель проекта

Создать полноценный одностраничный сайт на тему кино, используя современные технологии веб-разработки. В проекте была реализована адаптивная вёрстка, интерактивные элементы и удобный пользовательский интерфейс. Главная цель — показать навыки работы с HTML, CSS и JavaScript на практике.

---

## Используемые технологии

* HTML5
* CSS3
* JavaScript
* Flexbox
* Grid Layout
* LocalStorage
* Адаптивная вёрстка
* Модальные окна
* Валидация формы

---

## Реализованный функционал

* Шапка сайта с навигацией
* Каталог фильмов с карточками
* Кнопка «Показать ещё»
* Переключение светлой и тёмной темы
* Сохранение темы через localStorage
* Модальное окно с информацией о фильме
* Блок отзывов пользователей
* Форма обратной связи с валидацией
* Адаптация под мобильные устройства
